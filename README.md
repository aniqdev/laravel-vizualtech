To run swagger
```sh
git clone https://github.com/alijumaan/laravel-ecommerce.git
cd qberry-test/
composer install
npm install
npm run build
php artisan migrate --seed
php artisan serve
```
then go to [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
